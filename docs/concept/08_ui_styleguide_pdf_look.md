# UI Styleguide (PDF-Look, state-of-the-art)

## Ziel
Darstellung wie ein professionelles PDF:
- sehr gut lesbar
- scanbar
- klare Hierarchie
- minimalistische, sportlich-seriöse Ästhetik

---

## Design-Prinzipien
1) Information zuerst (Tabellen sind zentral)
2) Scanbarkeit (pro Woche in Sekunden erfassbar)
3) Konsistenz (Abstände/Typo/Tabellenstyle)
4) Druck-Anmutung (Seiten-Container)

---

## Layout
- Hintergrund: sehr helles Grau
- Seite: weiß, border + shadow, großzügiges Padding
- Header: Titel links, Stand/TM rechts
- Block: Titel + Kurztext + Tabelle + Regeln

---

## Typografie
- Font: Inter / System UI
- Titel: 18–22 px, bold
- Blocktitel: 16–18 px, semibold
- Text: 12–14 px
- Tabelle: 12–13 px, line-height 1.35–1.45

---

## Farbe
- 1 Akzentfarbe (z. B. Blau) für Linien/Headers
- Tabellenkopf leicht getönt
- Text nahezu schwarz, Sekundärtext dunkelgrau

---

## Tabelle
- klare Linien (1px)
- Kopfzeile tinted, bold
- erste Spalte “Woche” leicht getönt, bold
- Zellen: top aligned, genug Padding
- Inhalte als Bulletpoints

Format in Zellen:
- “Übung S×W @RPE/%TM: kg”
- z. B. “Bench 6×3 @80%TM: 65 kg”
- z. B. “Top Single 1×1 @RPE 8: Start 77.5 kg (±2.5)”

---

## Responsive
- Desktop: Tabelle
- Mobile: horizontales Scrollen ODER Cards/Tabs (optional)
